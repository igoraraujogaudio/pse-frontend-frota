Objetivo

Este orçamento tem como finalidade a prestação de serviços técnicos para a criação de um novo servidor, com instalação do sistema operacional Windows 11, e a migração dos sistemas e serviços atualmente utilizados, mantendo a estrutura centralizada em uma única máquina, conforme já ocorre no ambiente atual.

Principais Benefícios da Migração

🖥️ Sistema Atualizado

Windows 11 com todas as atualizações de segurança e performance mais recentes.

🔄 Migração Completa

Transferência de todos os sistemas sem perda de dados ou configurações.

🔒 Segurança Aprimorada

Configurações de firewall, permissões e acesso remoto seguro.

⚡ Performance Otimizada

Configuração otimizada para máximo desempenho dos serviços migrados.

Escopo Detalhado dos Serviços

2.1 Instalação do Sistema Operacional

Windows 11 - Configuração Completa

  • Instalação limpa do Windows 11 Professional
  • Configuração básica do sistema e drivers
  • Instalação de todas as atualizações críticas
  • Configuração de permissões e políticas de segurança
  • Criação de usuários administrativos e operacionais
  • Configuração de firewall e proteções básicas

2.2 Migração de Serviço PHP/Laravel

Ambiente Apache/PHP Completo

  • Instalação do Apache HTTP Server (versão mais recente estável)
  • Instalação do PHP com todas as extensões necessárias
  • Instalação e configuração do Composer
  • Migração completa do projeto Laravel existente
  • Configuração do arquivo .env com parâmetros corretos
  • Ajuste de permissões de diretórios (storage, cache, logs)
  • Configuração de Virtual Hosts no Apache
  • Testes completos de funcionamento e performance

2.3 Migração de Serviço em C#

IIS e ASP.NET Configuration

  • Instalação e configuração completa do IIS
  • Instalação do .NET Framework/Core necessário
  • Publicação do sistema C# (ASP.NET ou API)
  • Configuração de Application Pool otimizado
  • Ajuste de permissões de diretórios e usuários
  • Configuração de diretórios virtuais
  • Implementação de regras de URL Rewrite
  • Testes de funcionalidade e performance

2.4 Migração do Banco PostgreSQL

Database Migration Completa

  • Instalação do PostgreSQL (versão compatível)
  • Configuração de parâmetros de performance
  • Importação completa do backup (dump) do banco atual
  • Verificação de integridade dos dados migrados
  • Configuração de roles e permissões de usuários
  • Testes de conectividade com os sistemas
  • Configuração de backup automático
  • Otimização de índices e consultas

Ambiente Python Completo
